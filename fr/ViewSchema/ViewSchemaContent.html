Afficher le schéma


Voici un résumé du schéma que vous avez créé. Vous pouvez parcourir plusieurs langues (si vous en avez plusieurs) pour voir toutes les informations de schéma pour chaque langue.

Pour modifier l'une des entrées, utilisez le bouton Précédent pour revenir en arrière dans le rédacteur de schéma et apporter les modifications nécessaires.

Lorsque vous avez terminé et exporté votre schéma, vous créez deux fichiers. Le premier sera la version lisible par machine de votre schéma. Le second sera la version .txt lisible par l'homme de votre schéma. Nous vous suggérons de conserver ces deux fichiers ensemble afin que les machines et les humains puissent comprendre votre schéma.
