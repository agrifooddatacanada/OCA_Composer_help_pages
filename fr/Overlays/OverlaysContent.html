Superpositions - Ajout d'informations facultatives supplémentaires


Dans Overlays Capture Architecture (OCA), vous pouvez ajouter des fonctionnalités supplémentaires facultatives à votre schéma. Ce menu vous permet de sélectionner les informations supplémentaires que vous souhaitez écrire pour votre schéma. Vous pouvez ajouter, modifier et supprimer des fonctionnalités de schéma. La liste suivante inclut les superpositions supplémentaires qui peuvent être ajoutées manuellement au schéma.


Superpositions :

  • Codage des caractères : Documentez le codage des caractères des données pour chaque attribut.
  • Entrées obligatoires : Spécifiez si l'ensemble de données comporte des attributs pour lesquels des données sont obligatoires.
  • Règles de formatage : Spécifiez la règle de formatage qui s'applique aux données pour chaque attribut.
  • Cardinalité : Spécifiez le nombre exact, minimal ou maximal (ou minimal et maximal) d'entrées autorisées dans un enregistrement de données pour chaque attribut.
  • Encadrement des unités : Les unités déjà définies peuvent être mappées (encadrées) à l'ontologie d'unité UCUM. Cela permet aux utilisateurs de décrire leurs unités sans ambiguïté.
  • Informations de formulaire : Spécifiez comment chaque attribut et son champ de réponse correspondant apparaissent dans un questionnaire.
