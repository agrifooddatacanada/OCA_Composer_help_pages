Constructeur de Formulaire


Le Constructeur de Formulaire est une interface visuelle puissante de glisser-déposer qui vous permet de créer des formulaires interactifs pour la collecte de données. Organisez votre formulaire en pages et sections, et ajoutez des questions en faisant simplement glisser des attributs depuis la palette d'attributs.

Commencer

Lorsque vous ouvrez le Constructeur de Formulaire pour la première fois, vous verrez une interface propre et intuitive avec :

  • Onglets de Langue : Basculez entre différentes langues pour configurer des formulaires multilingues
  • Palette d'Attributs : Une liste complète d'attributs disponibles sur le côté gauche
  • Pages de Formulaire : L'espace de travail principal où vous construisez votre formulaire
  • Bouton Ajouter Page : Créer des pages supplémentaires pour votre formulaire

Travailler avec les Pages

Les pages sont les conteneurs principaux de votre formulaire. Chaque page peut contenir :

  • Des questions directes
  • Des sections (qui peuvent contenir plusieurs questions)

Ajouter une Nouvelle Page

  1. Cliquez sur le bouton "Ajouter Page" dans le coin supérieur droit
  2. La nouvelle page apparaîtra sous les pages existantes

Modifier une Page

  1. Cliquez sur l'icône d'édition (crayon) dans l'en-tête de la page
  2. Modifiez le titre de la page, l'étiquette de la barre latérale et la description
  3. Configurez ces éléments pour chaque langue dans votre schéma

Supprimer une Page

  1. Cliquez sur l'icône de suppression (poubelle) dans l'en-tête de la page
  2. Note : Vous ne pouvez pas supprimer la dernière page restante

Travailler avec les Sections

Les sections aident à organiser les questions liées au sein d'une page. Elles offrent une meilleure structure pour les formulaires complexes et améliorent l'expérience utilisateur.

Ajouter une Section

  1. Cliquez sur le bouton "Ajouter Section" dans une page
  2. La section apparaîtra comme un nouveau conteneur dans la page

Modifier une Section

  1. Cliquez sur l'icône d'édition dans l'en-tête de la section
  2. Modifiez le titre et la description de la section pour chaque langue

Supprimer une Section

  1. Cliquez sur l'icône de suppression dans l'en-tête de la section
  2. Toutes les questions dans la section seront également supprimées

Ajouter des Questions

Les questions sont créées en faisant glisser des attributs depuis la Palette d'Attributs vers vos pages de formulaire ou sections. Ce processus intuitif rend la construction de formulaires à la fois efficace et agréable.

Ajouter une Question

  1. Trouvez l'attribut souhaité dans la Palette d'Attributs (côté gauche)
  2. Faites glisser l'attribut vers une page ou section
  3. L'attribut deviendra automatiquement une question
  4. Note : Chaque attribut ne peut être utilisé qu'une seule fois dans tout le formulaire

Modifier une Question

  1. Cliquez sur l'icône d'édition sur n'importe quelle question
  2. Modifiez le titre de la question, le texte d'espace réservé et d'autres propriétés
  3. Configurez ces éléments pour chaque langue dans votre schéma
  4. Définissez si la question est obligatoire

Supprimer une Question

  1. Cliquez sur l'icône de suppression sur la question
  2. La question sera supprimée du formulaire

Opérations de Glisser-Déposer

Le Constructeur de Formulaire prend en charge plusieurs opérations intuitives de glisser-déposer :

Déplacer des Questions

  • Faites glisser des questions entre les pages
  • Faites glisser des questions entre les sections
  • Réorganisez les questions dans la même page ou section

Déplacer des Sections

  • Faites glisser des sections entre les pages
  • Réorganisez les sections dans la même page

Support Multilingue

Le Constructeur de Formulaire fournit un support multilingue complet :

  • Utilisez les onglets de langue pour basculer entre les langues
  • Configurez les titres de pages, les titres de sections et les étiquettes de questions pour chaque langue
  • Définissez le texte d'espace réservé pour chaque langue
  • Configurez les codes d'entrée (options de menu déroulant) pour chaque langue

Types d'Attributs

Différents types d'attributs créent différents types de questions, chacun optimisé pour des besoins spécifiques de collecte de données :

  • Attributs Texte : Créent des champs de saisie de texte ou des zones de texte
  • Attributs de Liste : Créent des menus déroulants avec des options prédéfinies
  • Attributs Date/Heure : Créent des sélecteurs de date et d'heure
  • Attributs Numériques : Créent des champs de saisie de nombres

Validation de Formulaire

Le Constructeur de Formulaire gère automatiquement la validation pour assurer l'intégrité des données :

  • Empêche l'utilisation en double d'attributs
  • Valide la structure du formulaire avant de continuer
  • Synchronise les données des questions avec la configuration principale du schéma

Conseils pour une Construction de Formulaire Efficace

  • Planifiez votre structure : Pensez aux regroupements logiques avant d'ajouter des questions
  • Utilisez des sections : Groupez les questions liées ensemble pour une meilleure organisation
  • Considérez l'expérience utilisateur : Placez les questions les plus importantes en premier
  • Testez le contenu multilingue : Basculez entre les langues pour vous assurer que tout le contenu est correctement configuré
  • Utilisez des étiquettes descriptives : Rendez les titres de questions et les espaces réservés clairs et utiles

Navigation

Utilisez les boutons de navigation en bas du Constructeur de Formulaire :

  • Retour : Retourner à l'overlay d'Informations de Formulaire
  • Suivant : Passer à la section Overlays avec votre formulaire terminé

Le Constructeur de Formulaire convertit automatiquement votre formulaire visuel au format d'overlay d'Informations de Formulaire approprié lorsque vous passez à l'étape suivante.
