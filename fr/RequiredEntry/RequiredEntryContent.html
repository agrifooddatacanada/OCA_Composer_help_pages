Entrée obligatoire

Les entrées obligatoires sont une superposition facultative pour votre schéma où vous pouvez spécifier si l'ensemble de données auquel le schéma fait référence doit avoir une entrée pour l'attribut spécifique.

Par exemple, si vous insistez pour que tous les enregistrements de l'ensemble de données nécessitent une valeur pour l'attribut « name », vous devez rendre « name » obligatoire ici. Lorsque les données sont validées par rapport à votre schéma, tous les enregistrements de données « name » vides dans l'ensemble de données peuvent être signalés comme des enregistrements non valides et traités en conséquence dans l'analyse.
