Form Builder


The Form Builder is a powerful visual drag-and-drop interface that allows you to create interactive forms for data collection. Organize your form into pages and sections, and add questions by simply dragging attributes from the attribute palette.

Getting Started

When you first open the Form Builder, you'll see a clean, intuitive interface with:

  • Language Tabs: Switch between different languages to configure multilingual forms
  • Attribute Palette: A comprehensive list of available attributes on the left side
  • Form Pages: The main workspace where you build your form
  • Add Page Button: Create additional pages for your form

Working with Pages

Pages are the main containers for your form. Each page can contain:

  • Direct questions
  • Sections (which can contain multiple questions)

Adding a New Page

  1. Click the "Add Page" button in the top-right corner
  2. The new page will appear below existing pages

Editing a Page

  1. Click the edit icon (pencil) in the page header
  2. Modify the page title, sidebar label, and description
  3. Configure these for each language in your schema

Deleting a Page

  1. Click the delete icon (trash) in the page header
  2. Note: You cannot delete the last remaining page

Working with Sections

Sections help organize related questions within a page. They provide better structure for complex forms and improve user experience.

Adding a Section

  1. Click the "Add Section" button within a page
  2. The section will appear as a new container within the page

Editing a Section

  1. Click the edit icon in the section header
  2. Modify the section title and description for each language

Deleting a Section

  1. Click the delete icon in the section header
  2. All questions within the section will also be removed

Adding Questions

Questions are created by dragging attributes from the Attribute Palette to your form pages or sections. This intuitive process makes form building both efficient and enjoyable.

Adding a Question

  1. Find the desired attribute in the Attribute Palette (left side)
  2. Drag the attribute to a page or section
  3. The attribute will automatically become a question
  4. Note: Each attribute can only be used once in the entire form

Editing a Question

  1. Click the edit icon on any question
  2. Modify the question title, placeholder text, and other properties
  3. Configure these for each language in your schema
  4. Set whether the question is required

Deleting a Question

  1. Click the delete icon on the question
  2. The question will be removed from the form

Drag and Drop Operations

The Form Builder supports several intuitive drag-and-drop operations:

Moving Questions

  • Drag questions between pages
  • Drag questions between sections
  • Reorder questions within the same page or section

Moving Sections

  • Drag sections between pages
  • Reorder sections within the same page

Multilingual Support

The Form Builder provides comprehensive multilingual support:

  • Use the language tabs to switch between languages
  • Configure page titles, section titles, and question labels for each language
  • Set placeholder text for each language
  • Configure entry codes (dropdown options) for each language

Attribute Types

Different attribute types create different question types, each optimized for specific data collection needs:

  • Text Attributes: Create text input fields or textareas
  • List Attributes: Create dropdown menus with predefined options
  • Date/Time Attributes: Create date and time pickers
  • Numeric Attributes: Create number input fields

Form Validation

The Form Builder automatically handles validation to ensure data integrity:

  • Prevents duplicate use of attributes
  • Validates form structure before proceeding
  • Syncs question data with the main schema configuration

Tips for Effective Form Building

  • Plan your structure: Think about logical groupings before adding questions
  • Use sections: Group related questions together for better organization
  • Consider user experience: Place the most important questions first
  • Test multilingual content: Switch between languages to ensure all content is properly configured
  • Use descriptive labels: Make question titles and placeholders clear and helpful

Navigation

Use the navigation buttons at the bottom of the Form Builder:

  • Back: Return to the Form Information overlay
  • Next: Proceed to the Overlays section with your completed form

The Form Builder automatically converts your visual form into the appropriate Form Information overlay format when you proceed to the next step.
